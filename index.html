Getting Started with MaxMSP

About

This is a compilation of sketches that provide a simplified way of connecting a visual programming (MaxMSP/Pure Data) language with an Arduino without any third-party externals or libraries. Instead of creating specialized examples of this connectivity idea, arduivis provides a modular solution by using design models that can be easily duplicated and implemented into many projects.

Requirements

  • Microcontroller (any type of Arduino)
    • Arduino
      • UNO, Leonardo, Mega 2560, Micro, Adafruit Trinket 5v, Adafruit Trinket Pro 5v
  • Visual Programming Language
    • MaxMSP
    • Max-For-Live
    • Pure Data Extended

    Getting Started

    • Download a visual programming language
      • MaxMSP
      • PD-Extended
    • Download the Arduino IDE
      • Arduino IDE
    • Download arduivis repository
      • arduivis
    • Connect the Arduino to the computer via a USB cable
      • Double check the Arduino is working properly by uploading the blink sketch
